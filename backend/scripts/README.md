# Analysis Scripts

This directory contains scripts for analyzing model behavior, intervention effectiveness, and risk reduction patterns.

## Scripts

### analyze_intervention_recommendations.py
Analyzes intervention effectiveness across different patient profiles to determine personalized recommendation logic.

**Usage:**
```bash
cd backend
python scripts/analyze_intervention_recommendations.py
```

### analyze_model_behavior.py
Analyzes model predictions and behavior patterns to ensure consistency and reliability.

**Usage:**
```bash
cd backend
python scripts/analyze_model_behavior.py
```

### analyze_risk_reduction.py
Analyzes risk reduction patterns across different interventions and patient types.

**Usage:**
```bash
cd backend
python scripts/analyze_risk_reduction.py
```

## Notes

- These scripts are for analysis and development purposes
- They are not part of the automated test suite
- Run from the `backend` directory to ensure proper path resolution
